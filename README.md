### Editors  
* [Neovim](https://neovim.io) — Базовый редактор.
* [LazyVim](https://www.lazyvim.org) — Дистрибутив конфига для Neovim.

### Kubernetes
* [k9s](https://github.com) — TUI для управления кластером (тема: TokyoNight).

### Modern CLI Tools
* [zoxide](https://github.com) — Умный переход по папкам (`cd`).
* [eza](https://github.com) — Современная замена `ls`.
* [bat](https://github.com) — Просмотр файлов с подсветкой (`cat`).
* [ripgrep](https://github.com) — Очень быстрый поиск текста (`rg`).
* [fd](https://github.com) — Быстрый поиск файлов.
* [fzf](https://github.com) — Интерактивный фильтр/поисковик.
