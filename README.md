### Editors  
* [Neovim](hhttps://github.com/neovim/neovim) — Базовый редактор.
* [LazyVim](https://github.com/LazyVim/LazyVim) — Дистрибутив конфига для Neovim.

### Kubernetes
* [k9s](https://github.com/derailed/k9s) — TUI для управления кластером (тема: TokyoNight).

### Modern CLI Tools
* [zoxide](https://github.com/ajeetdsouza/zoxide) — Умный переход по папкам (`cd`).
* [eza](https://github.com/eza-community/eza) — Современная замена `ls`.
* [bat](https://github.com/sharkdp/bat) — Просмотр файлов с подсветкой (`cat`).
* [ripgrep](https://github.com/BurntSushi/ripgrep) — Очень быстрый поиск текста (`rg`).
* [fd](https://github.com/BurntSushi/ripgrep) — Быстрый поиск файлов.
* [fzf](https://github.com/junegunn/fzf) — Интерактивный фильтр/поисковик.
